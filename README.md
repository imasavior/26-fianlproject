# 26組-finalproject
=======
對打遊戲

功能說明:
紅色與綠色玩家對戰，在時間結束後最高分者獲勝。
在8x8 LED 矩陣，用來顯示對戰畫面

7段顯示器顯示剩餘生命(從5開始)

當A攻擊B時，攻擊判定成功，5+=1。

當B攻擊A時，攻擊判定成功，5-=1。

當A攻擊B時，B使用格擋，攻擊判定失敗，不扣分。

當B攻擊A時，A使用格擋，攻擊判定失敗，不扣分。

當顯示器顯示到9，跑出藍色勝利 W。

當顯示器顯示到1，跑出紅色勝利 W。
